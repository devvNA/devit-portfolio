CV File Placeholder
====================

To enable the "Download CV" functionality:

1. Add your CV file as "cv-devit.pdf" to the /public folder
2. The download button in the navbar will automatically work
3. Make sure the filename matches exactly: cv-devit.pdf

Current location for CV: /public/cv-devit.pdf

The portfolio is configured to download from this location when users click the "Download CV" button in the navigation bar.
