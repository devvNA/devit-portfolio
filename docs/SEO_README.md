## ğŸ†˜ Troubleshooting

### Common Issues

**Q: Website not indexed after 2 weeks?**
```
Solutions:
1. Check robots.txt not blocking
2. Submit sitemap in Google Search Console
3. Request indexing manually
4. Check for technical errors
```

**Q: OG image not showing on social media?**
```
Solutions:
1. Verify image size (1200x630)
2. Check file size < 1MB
3. Clear cache in Facebook Debugger
4. Ensure image URL is accessible
```

**Q: Low SEO score in Lighthouse?**
```
Solutions:
1. Check meta description length
2. Add alt text to all images
3. Verify heading hierarchy (h1, h2, h3)
4. Test mobile responsiveness
```

**Q: Not ranking for keywords?**
```
Solutions:
1. Wait 1-3 months (SEO takes time)
2. Create more content
3. Build quality backlinks
4. Engage with community
5. Update content regularly
```
